**# 🧩 Sudoku Master - Interactive Web Game

¡Bienvenido a **Sudoku Master**! Este es un juego de Sudoku interactivo, moderno y totalmente funcional desarrollado para ser jugado directamente desde el navegador.

---

## 🚀 Características Principales
* **Generación de Tableros:** Lógica inteligente para crear desafíos.
* **Validación en Tiempo Real:** El sistema te avisa si hay errores en los números ingresados.
* **Cronómetro Integrado:** Mide tu tiempo y mejora tus récords.
* **Diseño Responsivo:** Optimizado para jugar tanto en computadoras como en dispositivos móviles.
* **Interfaz Moderna:** Estética limpia utilizando CSS3 avanzado.

## 🛠️ Tecnologías Utilizadas
Este proyecto fue construido utilizando estándares modernos de desarrollo web:
* **HTML5:** Estructura semántica del juego.
* **CSS3:** Estilos personalizados, animaciones y diseño adaptable.
* **JavaScript (ES6+):** Motor lógico del juego, manejo del tiempo y validaciones.
* **OpenCode AI:** Utilizado como asistente de desarrollo para optimizar la estructura modular y la eficiencia del código.

## 🎮 Cómo Jugar
1. Entra al sitio oficial: [Jugando Sudoku Master](https://juansilv19.github.io/Sudoku-Master/)
2. Elige una casilla vacía.
3. Ingresa un número del 1 al 9.
4. ¡Completa el tablero en el menor tiempo posible sin repetir números en filas, columnas o cuadrantes de 3x3!

## 📂 Estructura del Proyecto
* `index.html`: El corazón del juego.
* `style.css`: Toda la magia visual.
* `script.js`: La inteligencia detrás del tablero.
* `AGENTS.md`: Documentación de la asistencia de IA en el desarrollo.
